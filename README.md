# 🎯 错题本 · 高效复习（Error Book）

一款**零依赖、纯前端、手机优先**的错题本应用。直接双击 `index.html` 就能用，也可以部署到 GitHub Pages。

基于**艾宾浩斯遗忘曲线**+**本地持久化**设计，帮助学生高效管理错题、自动推送复习日程、一键组卷。

## ✨ 功能特性

### 📘 五大核心页面
| 页面 | 说明 |
|---|---|
| 🏠 **首页** | 错题总览（饼图/趋势图）、今日复习清单、快捷入口 |
| ✏️ **录入** | 题干、错误答案、正确答案、错误原因、多标签、难度星级、多图、📷拍照OCR |
| 📋 **题库** | 按科目/难度/状态筛选、搜索、详情、编辑、删除、收藏 |
| 🔁 **刷题** | 按科目顺序刷题、随机打乱、隐藏答案、掌握/再练、自动走艾宾浩斯周期 |
| 📊 **统计** | 科目分布饼图、高频知识点、近 4 周趋势、数据备份、错题组卷、艾宾浩斯自定义 |

### 🧠 智能复习
- 内置 **艾宾浩斯简易周期**（1/2/4/7/15/30 天，可自定义）
- 首页自动推送**今日复习清单**
- 错题连续掌握次数达标（默认 3 次）才标记"已掌握"
- 每个错题自动记录**出错次数**和**下次复习时间**

### 📤 数据自由
- **JSON 备份/导入**：支持在不同设备之间迁移
- **A4 排版打印**：一键全部打印、或**错题组卷**自由挑题生成专项试卷
- **完全离线**：所有数据仅保存在浏览器 localStorage，零上传

### 🎨 界面细节
- 🌓 深色 / 浅色 模式切换
- A+ 字体大小调节
- 手机优先布局（安全区适配刘海屏/底部横条）
- 响应式，桌面端自动扩展
- 流畅的卡片动画和过渡

## 🚀 快速开始

### 方式一：本地直接打开
```
双击 index.html 即可使用（建议用 Chrome / Edge / Safari）
```

### 方式二：本地启动静态服务器
```bash
# Python
python -m http.server 8080

# Node
npx serve .
```
然后浏览器访问 `http://localhost:8080`。

### 方式三：部署到 GitHub Pages
1. Fork 本仓库
2. Settings → Pages → Source 选择 `main` 分支 `/ (root)`
3. Save → 等 1 分钟即可通过 `https://<你>.github.io/错题本/` 访问

也可以用 Cloudflare Pages / Vercel / Netlify 等免费平台，零配置部署。

### 手机端访问
部署后，在手机浏览器打开即可；或者在本地局域网启动服务器后，手机访问 `http://<电脑IP>:8080`。

## 📁 目录结构

```
错题本/
├── index.html            # 入口
├── src/
│   ├── styles.css        # 全部样式（含深色模式、打印样式）
│   ├── core.js           # 数据层：localStorage、艾宾浩斯、种子题库
│   └── router.js         # 路由 + 五大页面视图 + 交互逻辑
├── .gitignore
└── README.md
```

整个项目**零 NPM 依赖、零构建过程**。所有代码分在 3 个 JS 文件 + 1 个 CSS 文件，方便阅读和二次开发。
